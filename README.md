# SSB MCP Server

MCP-server som gir tilgang til SSBs statistikkdata via naturlig språk.

## Koble til i Claude.ai

1. Gå til Settings → Integrations → Add custom integration
2. Lim inn URL: `https://DIN-RAILWAY-URL/sse`

## Verktøy

- **search_ssb** – søk etter tabeller på norsk
- **get_table_metadata** – se variabler og koder for en tabell
- **query_ssb_table** – hent faktiske data

## Datakilde

SSB PxWebApi v2 – åpent API, ingen nøkkel nødvendig.
